# Acceptance criteria

This addon pack is ready when:

- every added skill has `SKILL.md`
- every skill contains evidence-first behavior
- every skill contains safe workflow behavior
- every skill contains anti-patterns
- every skill contains token-saving tip
- validation test passes
- follow-up roadmap is clear

Validation command:

```bash
python3 tests/test_admin_addon_skill_pack.py
```
