# Admin addon skill pack source basis

Research sources used for the first addon wave include:

- GNU grep manual and ripgrep guide for token-saving evidence patterns
- Google shell style guide, GNU Bash manual, and ShellCheck for script standards
- Linux kernel memory docs, `free(1)`, and `swapon(8)` for memory/swap semantics
- rsync manual, GNU tar manual, and NIST contingency planning for backup/restore discipline
- OpenSSH sshd_config manual for SSH hardening
- logrotate manual for rotation behavior and copy-truncate risk
- auditd/auditctl/audit rules manuals for audit design
- rsyslog official docs for rulesets, actions, queues, and forwarding
- tcpdump and pcap-filter manuals for bounded packet capture
- LVM manuals for LV growth/reduction/snapshots/thin pools
- NGINX, PHP-FPM, and MySQL official docs for service-stack experts
- NIST incident response guidance for incident workflow design

Each skill should be expanded later with a dedicated `research-sources.md` file.
