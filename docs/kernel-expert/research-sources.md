# Kernel Expert — Research Sources

Use official man pages, upstream project docs, kernel/systemd documentation, and distro community references before applying production changes. This module was designed around read-only first diagnosis, exact config ownership, validation, rollback, and refusal of destructive shortcuts.

Preferred live-research sources:

- man7.org Linux man pages for command semantics.
- systemd man pages for timers, timedatectl, mount units, and transient jobs.
- chrony-project.org documentation for chronyd and chronyc behavior.
- docs.kernel.org for kernel command-line parameters, kdump, modules, filesystems, and admin guides.
- Distribution docs and community pages for distro-specific defaults: Red Hat, Debian, Ubuntu, Fedora, Rocky, AlmaLinux.
