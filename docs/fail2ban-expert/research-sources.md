# fail2ban-expert research sources

Primary references:

- Fail2Ban upstream README for purpose, firewall rule updates, and warning that Fail2Ban reduces bad authentication attempts but does not remove weak-authentication risk.
- Upstream `jail.conf` for default options, local override guidance, `bantime`, `findtime`, `maxretry`, backends, `ignoreip`, and actions.
- Upstream filter examples for service-specific filters.
- `fail2ban-client` man page and operational practice for status, get/set, unban, and debug output.
