# networking-expert research sources

Primary references used:

- `ip-route(8)` for routing table management, route selection, route types, metrics, and policy routing concepts.
- `ip-address(8)` and `ip-link(8)` for addresses, interfaces, link details, and runtime state.
- `ss(8)` for socket inspection and TCP state analysis.
- `tc(8)` for traffic control/qdisc inspection.
- `ethtool(8)` for NIC/driver/offload/ring/channel inspection.
- Linux kernel networking sysctl documentation for TCP/core tunables.
- Field SRE practice: path-based diagnosis, no remote network state change without rollback, and persistent configuration through the active network manager.

Check local versions and active network manager before applying commands.
