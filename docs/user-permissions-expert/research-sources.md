# Research sources

Primary references used:

- `useradd(8)` and `usermod(8)` for account creation/modification behavior.
- `sudoers(5)` and `visudo(8)` for sudo policy and validation behavior.
- `limits.conf(5)` for PAM-session limits context.
- `getent(1)`, `passwd(1)`, `chage(1)` common admin references.
