# Research sources

Primary references used while designing this module:

- Red Hat RHEL in-place upgrade documentation and Leapp pre-upgrade report workflow.
- PostgreSQL `pg_upgrade` documentation for major-version migration behavior.
- MySQL `mysqldump` documentation for logical backup/transfer behavior.
- rsync man page for archive mode, dry-run, delete, SSH transport, and trailing slash behavior.
- DNF man page for package operations, advisories, download-only, history, and upgrade behavior.
- OpenSSH `sshd_config` and `sftp` documentation for SFTP/chroot and secure transfer behavior.
- Existing plugin experts: systemd, sysctl, limits, networking, firewall, Fail2Ban, command, user/file/ACL permissions.
