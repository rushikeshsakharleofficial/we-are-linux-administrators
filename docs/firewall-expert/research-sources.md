# firewall-expert research sources

Primary references:

- firewalld documentation and `firewall-cmd` manual for zones, runtime/permanent behavior, reload behavior, and rich/direct rules.
- nftables wiki and `nft(8)` practice for rulesets, tables, chains, sets, and atomic file workflow.
- `iptables(8)`, `iptables-save(8)`, and `iptables-restore(8)` manual pages for legacy/compatibility rule inspection and restore.
- Ubuntu `ufw(8)` manual for UFW syntax, dry-run, remote management warnings, app profiles, logging, and reports.
- Netfilter operational practice: backup first, identify owner/backend, smallest scoped rule, rollback validation.
