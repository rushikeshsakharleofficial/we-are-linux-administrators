12 Real Scenarios

Common Linux problems and exactly how to approach them with linux-admin. Each shows the right skill, investigation method, and expected answer.

1
Nginx service failed after config change
Skill /linux-admin:service
Method journalctl -xe → nginx -t → find bad directive
Answer Exact bad line in config + corrected directive
/linux-admin:service nginx.service failed after config change
2
Server can ping IP but DNS fails
Skill /linux-admin:network
Method check resolv.conf → test stub resolver → trace systemd-resolved
Answer Root cause + fix command
/linux-admin:network can ping 8.8.8.8 but DNS fails
3
Disk full but du and df don't match
Skill /linux-admin:filesystem-expert
Method df -h → du -sh → lsof +L1 → find deleted-open file
Answer PID holding file + kill/HUP command
/linux-admin:filesystem-expert /var full but du shows only 60%
4
PostgreSQL "too many open files"
Skill /linux-admin:limits-expert
Method systemctl show | grep LimitNOFILE → ulimit → pam.d
Answer Correct LimitNOFILE in systemd drop-in
/linux-admin:limits-expert postgres FATAL: too many open files
5
Fail2Ban not banning SSH attempts
Skill /linux-admin:fail2ban-expert
Method fail2ban-client status sshd → check filter regex → test regex
Answer Fixed filter regex or missing log path
/linux-admin:fail2ban-expert SSH brute-force not being banned
6
Firewall rule blocking HTTPS after deploy
Skill /linux-admin:firewall-expert
Method nft list ruleset → trace HTTPS chain → find deny rule
Answer Exact blocking rule + lockout-safe fix
/linux-admin:firewall-expert lockout-safe HTTPS blocked after deploy
7
fstab entry causing boot failure
Skill /linux-admin:disk-mounting-expert
Method findmnt --verify → check UUID → check fstype → validate options
Answer Corrected fstab line with explanation
/linux-admin:disk-mounting-expert validate fstab before reboot
8
Kernel panic after kernel upgrade
Skill /linux-admin:kernel-expert
Method dmesg → crash dump → call trace → identify module
Answer Module name + rebuild/rollback command
/linux-admin:kernel-expert triage kernel panic after upgrade
9
SFTP migration with users and keys
Skill /linux-admin:migration-expert
Method inventory users/keys → plan transfer → validate chroot jails
Answer Ordered migration runbook
/linux-admin:migration-expert SFTP server migration preserve chroot jails
10
Broken APT/DNF transaction
Skill /linux-admin:package-manager-expert
Method apt-get -f install → dpkg --configure -a → check lock files
Answer Exact fix sequence for the transaction state
/linux-admin:package-manager-expert apt transaction interrupted
11
Cloudflare DNS audit
Skill /linux-admin:cf-expert
Method check A/AAAA → proxy status → TTLs → page rules
Answer DNS audit report with findings
/linux-admin:cf-expert audit DNS for example.com
12
Chrony/NTP clock drift
Skill /linux-admin:chrony-expert
Method chronyc tracking → chronyc sources -v → check pool reachability
Answer Root cause + chrony.conf fix
/linux-admin:chrony-expert clock drifting 200ms per hour
